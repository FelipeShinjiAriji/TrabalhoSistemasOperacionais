Para chamar a execução do programa, no terminal entre na pasta Code do arquivo e execute o seguinte comando:
	python Main.py <address> <quadro> '<algorítmo>'
Exemplo no Windows: 
	cd C:\Users\Usuario\Documents\SistemasOperacionais\Prova\Prova2\PythonCode\TrabalhoSistemasOperacionais\Code
	python Main.py C:\Users\Usuario\Documents\SistemasOperacionais\Prova\Prova2\ArquivoLivro2\addresses.txt 128 'LRU'
